NightLifeSG

An Android Application that provides touristic support regarding the night-life in Singapore. It aims to display location info as well as calculate the best route to the selected tourist spots (via Taxi, Singapore MRT and Walking). To calculate the best route, the application also takes in the user inputted cost constraint value (Traveling Salesman Problem).

Skills:

  • Android
  • Algorithms
  • Google Maps API
